About

身為二十一世紀的機械設計工程師, 由 Dart 入門, 以 Python 建立設計抽象模型 Prototypes, 採用 C 執行機電控制, 然後在 Javascript 的網際與 Flutter 的手機介面中, 運行 Cython 以及 N-API 轉出的核心 C++ 編譯碼, 應該成為基本功.

計算機程式建議學習順序

1 ~ 4 為大一上計算機程式課程內容

1. 利用 Dartpad 學習基本 Dart 語法與 Flutter 應用

配合 Quick Start Guide to Dart Programming (2020) 與 Beginning App Development with Flutter (2019) 電子書章節說明

2. 在 Dartpad 中導入 HTML 與 CSS, 利用 Dart 結合 HTML5 Canvas 繪圖

參考 HTML5 Programmer’s Reference (2015) 電子書

3. 導入 Dart SDK, 在 SciTE 編輯器中執行  Dart 程式

參考 SciTE 執行 Dart 程式設定

4. 利用 Dart SDK 中的 dart2js 將 Dart HTML 程式轉為網際程式

參考  dart2js 編譯器說明

分組程式任務執行示範網站: https://mdecourse.github.io/cp2019g0 

5 ~ 6 為大一下網際內容管理課程內容

5. 導入 Flutter SDK, 在 Dartpad Embedded 環境中練習 Flutter 程式語法

參考 Flutter 按鈕中的程式說明

6. 利用 Flutter 網際與手機程式作為 GUi 介面, 與遠端的 Flask Python 程式互動

參考 HTML 程式中的 Python 按鈕與配合的 Flask 程式碼

7 ~ 9 為大二上電腦輔助設計實習課程內容

7. 利用 Cython 語法編寫遠端 Flask Python 演算程式庫, 透過 MSYS2 C++ 編譯為動態連結程式庫

參考 Pyslvs 中的機構分析與合成核心程式庫原始碼

8. Flask Python 結合 C++ 動態連結程式庫, 與 Flutter 網際與手機程式互動

9. 在 Dart 網際程式中導入 Javascript 程式庫, 與後端的 Flask Python 伺服程式互動

10 ~ 12 為大二下協同產品設計實習課程內容

10. 利用後端的 NXOpen for Python 編寫 3D 零組件二次開發 Journal 程式, 與 Flutter 互動

參考 NX12 中的 NXOpen for Python 程式碼

11. 利用 MSYS2 修改編譯 Solvespace C++ 程式, 加入 WRL 零組件轉出功能

參考 Solvespace 編譯流程說明

12. 將 NX 或 Solvespace 轉出的零組件檔案, 轉入 Webots, 利用 Flutter、Python 與 C/C++ 進行機電資虛實系統整合設計、模擬與控制
