Mac Notebook

初學者參考手冊:

1. Mac OS X for Absolute Beginners

https://link.springer.com/book/10.1007/978-1-4842-1913-3

2. Quick Start Guide to Dart Programming (2020)

https://link.springer.com/book/10.1007/978-1-4842-5562-9

3. Practical Flutter (2019)

https://link.springer.com/book/10.1007/978-1-4842-4972-7

4. Beginning App Development with Flutter (2019)

https://link.springer.com/book/10.1007/978-1-4842-5181-2

為 Python3 安裝 pip:

利用 python3 python3 與 get-pip.py 安裝 pip.

sudo python3 get-pip.py

利用 Mac App Store 安裝 XCode:

https://apps.apple.com/tw/app/xcode/id497799835?mt=12

安裝 CMSiMDE 所需模組:

利用 pip 安裝 flask, bs4, lxml, pelican, markdown, flask_cors, leo

sudo pip install flask bs4 lxml pelican markdown flask_cors leo

安裝 Homebrew:

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

利用 Homebrew 安裝 OpenJDK:

brew tap AdoptOpenJDK/openjdk
brew cask install adoptopenjdk8

安裝 Flutter:

https://flutter.dev/docs/get-started/install/macos

利用 Homebrew 安裝 VSCode:

brew update
brew cask install visual-studio-code

透過 XCode 使用 iOS Simulator:

利用  Xcode ➤ Open Developer Tool ➤ Simulator 開啟 iOS Simulator

Flutter iOS 手機程式開發:

開啟 VSCode 與 iOS Simulator

利用下拉式表單 Code ➤ Preferences ➤ Extensions 搜尋 Flutter Extensions 並完成安裝

利用 Command + Shift + P 開啟 Project 表單, 選擇 Flutter: New Project 建立專案

選擇儲存 Flutter Project 的目錄

Counter Demo 專案開啟後, 利用下拉式表單 Debug ➤ Run Without Debugging  完成專案編譯, 並在 iOS Simulator 手機中執行.
