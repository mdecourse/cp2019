Windows Notebook

上課學員登入 @gm 電子郵箱後, 所下載的 dartpad_based.7z 與 flutter_portable_final.7z 適用於在公共的 Windows 10 64 位元電腦上使用.

針對使用 Windows 筆記型電腦上課的學員, 則可以自行安裝 Python, 7zip, SciTE, Portable, OpenJDK, Flutter SDK, ShareX, Process Explorer, Zoom it, Tiny C 以及 Visual Studio Code 等套件.

並且利用 pip 安裝 flask, bs4, lxml, pelican, markdown, flask_cors, leo 等模組.

安裝 Git:

https://git-scm.com/download/win

安裝 Python3 (含 pip):

https://www.python.org/downloads/

安裝 CMSiMDE 所需模組:

pip install flask bs4 lxml pelican markdown flask_cors leo

安裝 OpenJDK:

https://java.tutorials24x7.com/blog/how-to-install-openjdk-12-on-windows

安裝 Flutter:

https://flutter.dev/docs/get-started/install/windows

安裝  Android-SDK-Tools:

https://developer.android.com/studio#downloads

安裝 Bluestacks:

https://www.bluestacks.com/

安裝 VSCode:

https://code.visualstudio.com/download

開啟 VSCode  後, 利用下拉式表單 Code ➤ Preferences ➤ Extensions 搜尋 Flutter Extensions 並完成安裝

Flutter Android 手機程式開發:

開啟 VSCode 與 Bluestacks

利用 Android SDK platform-tools 目錄中的 adb.exe, 執行

 adb connect localhost:5555

讓電腦中的 VSCode Flutter 手機程式專案, 可以透過內定的 5555 埠號, 將執行結果呈現在 localhost 電腦開啟的 Bluestacks 畫面中.

在 VSCode  套件中, 利用 Ctrl + Shift + P 開啟 Project 表單, 選擇 Flutter: New Project 建立專案

選擇儲存 Flutter Project 的目錄

Counter Demo 專案開啟後, 利用下拉式表單 Debug ➤ Run Without Debugging  完成專案編譯, 並在 Bluestacks 手機介面中執行.
